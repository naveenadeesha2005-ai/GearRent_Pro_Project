# GearRentPro Management System

## Project Description
A JavaFX application for managing equipment rentals, branches, and customer memberships.

## Database Configuration
1. Import the SQL file located in `/sql/gearrent_backup.sql` using MySQL Workbench.
2. Ensure your local MySQL service is running on port 3306.
3. Update `DatabaseConnection.java` with your local username and password.

## How to Run
1. Clone the repo: `git clone <your-url>`
2. Open in NetBeans (Maven project).
3. Run `Clean and Build`.
4. Run the project using the `Run` button or `mvn javafx:run`.
